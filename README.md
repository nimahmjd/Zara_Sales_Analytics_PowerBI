# Zara Sales Analytics Dashboard (Power BI)

## Project Overview
This project analyzes Zara product sales data to understand revenue drivers,
promotion effectiveness, and category performance using Power BI.

## Key Insights
- Promotions contribute ~50% of total revenue
- Jackets are the highest revenue-generating category
- Sweaters and T-shirts are more promotion-sensitive
- Revenue is not overly dependent on discounts

## Tools Used
- Power BI
- DAX (SUMX, CALCULATE)
- Data modeling & visualization
